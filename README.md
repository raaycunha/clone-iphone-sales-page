# iPhone 13 Pro Clone - Apple Page

Este projeto é uma reprodução da interface de vendas do iPhone 13 Pro, focada em fidelidade visual e interatividade. O objetivo principal foi consolidar fundamentos de **JavaScript Puro** e **CSS Avançado**, preparando o terreno para futuros estudos em **React** e **TypeScript**.

## 🚀 Tecnologias Utilizadas

*   **HTML5**: Estrutura semântica para melhor acessibilidade.
*   **CSS3**: Estilização moderna, uso de Flexbox, Media Queries e transições suaves.
*   **JavaScript (ES6+)**: Manipulação de DOM, tratamento de eventos (`change`, `mousedown`, `mouseup`) e lógica de carregamento dinâmico.

## 📱 Funcionalidades

- **Troca Dinâmica de Cores**: Ao selecionar uma cor, a imagem do produto é atualizada com um efeito de *fade-in*.
- **Feedback Tátil nos Botões**: Efeito visual de "clique" (scale) nos botões de compra.
- **Design Responsivo**: Interface adaptável para diferentes tamanhos de tela.
- **Pré-carregamento de Imagens**: Uso do evento `onload` para garantir que a imagem só apareça quando estiver totalmente carregada, evitando "pulos" na tela.

## 🧠 Aprendizados

Este projeto foi fundamental para entender:
1. A importância do evento `change` em elementos de formulário (`radio buttons`).
2. Como gerenciar estados de interface apenas com JS Puro.
3. Manipulação de propriedades de estilo e classes via script.
